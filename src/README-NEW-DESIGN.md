4. **生成代码**

## 🔧 自定义和调整

### 修改颜色
在CSS中搜索并替换：
- `#8B9D83` - 主题绿色
- `#333` - 文字颜色
- `rgba(255, 255, 255, 0.95)` - 导航栏背景

### 修改字体
在CSS中修改：
```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", ...
```

### 调整间距
修改以下值：
- `gap: 40px` - 图片间距
- `margin-bottom: 80px` - 区块间距
- `padding: 120px 40px` - 页面内边距

## 📱 响应式断点

- **桌面**：> 1024px
  - 三列网格
  - 显示侧边装饰
  
- **平板**：768px - 1024px
  - 两列网格
  - 隐藏侧边装饰
  
- **手机**：< 768px
  - 单列布局
  - 简化导航
