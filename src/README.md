## 💡 常见问题

**Q: 图片不显示？**  
A: 确保`new/SP网页图片素材/`文件夹存在且路径正确。

**Q: 如何修改颜色？**  
A: 在CSS中搜索`#8B9D83`（主题绿色）并替换。

**Q: 如何添加新项目？**  
A: 复制现有项目页面，修改内容和图片路径。

**Q: 支持手机访问吗？**  
A: 是的，所有页面都是响应式设计。
