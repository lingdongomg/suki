在CSS中搜索并替换：

```css
/* 主题绿色 */

/* 文字颜色 */
#333

rgba(255, 255, 255, 0.95)
```

### 修改字体

```css
font-family: -apple-system, BlinkMacSystemFont, 
             "Segoe UI", "PingFang SC", 
             "Hiragino Sans GB", "Microsoft YaHei", 
             sans-serif;
```

### 调整间距

```css
/* 图片间距 */
gap: 40px;

/* 区块间距 */
margin-bottom: 80px;

/* 页面内边距 */
padding: 120px 40px 80px;
```

### 修改动画速度

```css
/* 淡入动画 */
animation: fadeInUp 0.8s ease-out;

/* 悬停过渡 */
transition: all 0.3s;
```

## 🐛 常见问题

### Q: 图片不显示？
**A:** 检查以下几点：
1. 确保 `new/SP网页图片素材/` 文件夹存在
2. 检查图片路径是否正确
3. 确认图片文件名大小写匹配

### Q: 布局错乱？
**A:** 
1. 使用现代浏览器（Chrome、Firefox、Safari、Edge）
2. 清除浏览器缓存
3. 检查浏览器窗口大小

### Q: 如何修改项目顺序？
**A:** 在 `index.html` 中调整目录区域的项目卡片顺序

### Q: 如何添加新项目？
**A:** 
1. 复制现有项目页面
2. 修改标题和图片路径
3. 在 `index.html` 目录中添加新卡片

### Q: 动画不流畅？
**A:**
1. 检查图片大小（建议 < 500KB）
2. 关闭浏览器的硬件加速
3. 更新浏览器到最新版本
