Hi 我们是Suki+Partners 思和朋友们

思和朋友们 Suki+Partners 是一群多元设计创作者，创作范围主要是婚礼定制与婚礼策划，包括空间装置与花艺设计。服务覆盖全国及海外。

专注于小众婚礼、自然婚礼、诗意婚礼策划与设计。我们通过视觉呈现传递你们的感受，并帮助建立对话。

我们希望探索多界融合的新领域，探寻个人特征与仪式的原始意义。空间、花艺、建筑、场域、装置、平面传达等被整合为整体，以多种材质与形态来创作、传达信息，寻找与空间精神相契合的表达，保持生野。
